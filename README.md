# "Hangman" game

A classic word game implemented in Java 25 with a console interface.  
Got the idea from this roadmap https://zhukovsd.github.io/java-backend-learning-course/projects/hangman/

---

**Project is meant to run from an IDE**

---

1. Clone the project via

```cmd
git clone https://github.com/J03MAMAD3ZZ/Hangman.git
```
2. Navigate to **`src/main/java/hangman`**

3. Run the program through **`Main.java`**
